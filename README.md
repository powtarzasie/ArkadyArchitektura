# Arkady Architektura and Design

Strona WWW marki **Arkady Architektura and Design** — architektura wnętrz, Katarzyna Janicka‑Świerguła (Łódź).
Ponadczasowa, edytorska estetyka premium: krem/greige, mosiężne akcenty, Cormorant Garamond + Jost.

## Struktura

- `index.html` — opublikowana strona (kopia pliku źródłowego, serwowana przez GitHub Pages).
- `Arkady Architektura.dc.html` — plik źródłowy (Claude Design Component, framework `x-dc`).
- `support.js` — runtime renderujący komponenty `x-dc`.
- `assets/` — logotypy, zdjęcia realizacji oraz portret.

## Podgląd lokalny

```bash
python -m http.server 8742
# następnie otwórz http://localhost:8742/
```
